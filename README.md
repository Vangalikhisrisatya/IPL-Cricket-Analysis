# IPL Cricket Analysis using Python

## Project Overview

This project analyzes IPL cricket data using Python to discover insights into team performance, player statistics, match outcomes, and venue trends through data analysis and visualization.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- matches.csv
- deliveries.csv

## Analysis Performed
- Team-wise performance analysis
- Top run scorers
- Top wicket takers
- Toss decision analysis
- Venue analysis
- Season-wise statistics
- Data visualization using charts and graphs

## Files
- IPL Cricket Analysis.ipynb
- matches.csv
- deliveries.csv
